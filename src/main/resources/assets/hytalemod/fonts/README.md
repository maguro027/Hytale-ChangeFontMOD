# Noto Sans JP フォント

このMODは **Noto Sans JP** (Google Fonts) を使用しています。

## 含まれているファイル

- `custom.ttf` - Noto Sans JP (Variable Font)
- `NotoSansJP-Regular.ttf` - オリジナルファイル

## フォントの変更方法

お好みのフォントに変更したい場合：

1. お好みのTTFフォントファイルを用意
2. `custom.ttf` を上書き
3. MODを再ビルド

## フォント情報

- **フォント名**: Noto Sans JP
- **提供元**: Google Fonts
- **ライセンス**: SIL Open Font License 1.1
- **URL**: https://fonts.google.com/noto/specimen/Noto+Sans+JP

## 対応フォーマット

- TrueType Font (.ttf)
- OpenType Font (.otf も custom.ttf にリネームすれば使用可能)
